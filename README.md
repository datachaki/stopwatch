# stopwatch
A high-visibility, centered terminal stopwatch written in Bash. It features large ASCII art and interactive color switching.


Before running `stopwatch`, ensure you have the following installed:

1.  **procps** 
2.  **ncurses** 
3.  **figlet**


Installation

```bash
git clone https://github.com/datachaki/stopwatch
cd stopwatch
chmod +x install.sh
./install.sh
